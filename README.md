# alyra-projet4-nft

## setup

- duplicate the `.env.example` to `.env` and fill your values
- install dependancies
```
npm install
```
- deploy to ganache 
```
truffle migrate --reset --network ganache
```
- build the client
```
cd client
cp .env.example .env
## fill your own values then
npm install
npm start
```
